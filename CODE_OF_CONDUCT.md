# Code of Conduct

This is a small, personal project, but the same basic rules apply as anywhere else:

- Be respectful. Disagree about code or content, not about people.
- No harassment, insults, or discriminatory language or behavior, in issues, PRs, or
  anywhere else related to this project.
- Assume good faith most mistakes are honest ones.

Reports of unacceptable behavior can be sent privately to the maintainer via the contact
links on [krystianpinczak.com](https://www.krystianpinczak.com). Maintainers may remove
comments, commits, code, issues, or PRs that don't follow this Code of Conduct, and may
ban contributors for repeated or serious violations.
